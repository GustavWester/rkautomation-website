# RK Automation Website

Frontend codebase for the RK Automation website.  
A lightweight, static web application showcasing AI-powered invoice automation with optional n8n API integration.

---

## Overview

This project is a **pure frontend** application built with vanilla HTML, CSS, and JavaScript.  
It is designed to be framework-free, fast, and easy to host on any static web server.

Key goals:
- Clean, modern UI
- Responsive layout
- Simple integration point for backend automation (n8n)
- No build step required

---

## Tech Stack

- **HTML5**
- **CSS3** (custom properties, gradients, responsive layout)
- **Vanilla JavaScript**
- **Static hosting compatible**

No frameworks, bundlers, or transpilers are required.
